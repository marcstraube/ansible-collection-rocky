# Changelog

All notable changes to this collection will be documented in this file.

## [1.0.0] - 2026-04-10

### Summary

Initial public release of the `marcstraube.rocky` collection.

Automated Rocky Linux server installation via Kickstart netinstall
with post-install configuration. Supports Rocky Linux 9 and 10.

### Roles

- **installer** - Kickstart file generation and post-install configuration
